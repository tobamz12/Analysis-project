#1.0-Introduction
This is a dashboard contains railway ticket sales dataset consisting of information on ticket
purchases, journeys, delays, prices, payment methods, and customer attributes. The task was to clean the dataset, perform structured analysis, and present clear business insights

#2.0-Data Source
The Data source for this challenge was provided by UTIVA, the company who hosted the Challenge. It was in an Excel format.
